    Detailed Description¶

    The public libopentracecapture header file to be used by frontends.

    This is the only file that libopentracecapture users (frontends) are supposed to use and #include. There are other header files which get installed with libopentracecapture, but those are not meant to be used directly by frontends.

    The correct way to get/use the libopentracecapture API functions is:

    #include <libopentracecapture/libopentracecapture.h>

    enum otc_mqflag¶

    Enumerator Value Description
    OTC_MQFLAG_AC 0x01 Voltage measurement is alternating current (AC).
    OTC_MQFLAG_DC 0x02 Voltage measurement is direct current (DC).
    OTC_MQFLAG_RMS 0x04 This is a true RMS measurement.
    OTC_MQFLAG_DIODE 0x08 Value is voltage drop across a diode, or NAN.
    OTC_MQFLAG_HOLD 0x10 Device is in "hold" mode (repeating the last measurement).
    OTC_MQFLAG_MAX 0x20 Device is in "max" mode, only updating upon a new max value.
    OTC_MQFLAG_MIN 0x40 Device is in "min" mode, only updating upon a new min value.
    OTC_MQFLAG_AUTORANGE 0x80 Device is in autoranging mode.
    OTC_MQFLAG_RELATIVE 0x100 Device is in relative mode.
    OTC_MQFLAG_SPL_FREQ_WEIGHT_A 0x200 Sound pressure level is A-weighted in the frequency domain, according to IEC 61672:2003.
    OTC_MQFLAG_SPL_FREQ_WEIGHT_C 0x400 Sound pressure level is C-weighted in the frequency domain, according to IEC 61672:2003.
    OTC_MQFLAG_SPL_FREQ_WEIGHT_Z 0x800 Sound pressure level is Z-weighted (i.e. not at all) in the frequency domain, according to IEC 61672:2003.
    OTC_MQFLAG_SPL_FREQ_WEIGHT_FLAT 0x1000 Sound pressure level is not weighted in the frequency domain, albeit without standards-defined low and high frequency limits.
    OTC_MQFLAG_SPL_TIME_WEIGHT_S 0x2000 Sound pressure level measurement is S-weighted (1s) in the time domain.
    OTC_MQFLAG_SPL_TIME_WEIGHT_F 0x4000 Sound pressure level measurement is F-weighted (125ms) in the time domain.
    OTC_MQFLAG_SPL_LAT 0x8000 Sound pressure level is time-averaged (LAT), also known as Equivalent Continuous A-weighted Sound Level (LEQ).
    OTC_MQFLAG_SPL_PCT_OVER_ALARM 0x10000 Sound pressure level represented as a percentage of measurements that were over a preset alarm level.
    OTC_MQFLAG_DURATION 0x20000 Time is duration (as opposed to epoch, ...).
    OTC_MQFLAG_AVG 0x40000 Device is in "avg" mode, averaging upon each new value.
    OTC_MQFLAG_REFERENCE 0x80000 Reference value shown.
    OTC_MQFLAG_UNSTABLE 0x100000 Unstable value (hasn't settled yet).
    OTC_MQFLAG_FOUR_WIRE 0x200000 Measurement is four wire (e.g. Kelvin connection).

    Values for otc_analog_meaning.mqflags.
