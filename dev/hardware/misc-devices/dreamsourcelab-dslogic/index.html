    Firmware In order to use this device with OpenTraceCapture the vendor firmare and bitstream files (v0.97) are required. ### Installing firmware files via a bash script The easiest method to install those is to use the OpenTraceLab-fwextract-dreamsourcelab-dslogic script. It will download the correct files, rename them to the correct filenames as expected by OpenTraceCapture and install them. Example usage: $ PREFIX=$HOME/sr ./OpenTraceLab-fwextract-dreamsourcelab-dslogic This will install the files into \$HOME/sr/share/OpenTraceLab-fimware. Without PREFIX, the files will be installed into /usr/local/share/OpenTraceLab-firmware by default. ### Installing firmware files manually If you want to avoid using the script, you can manually
