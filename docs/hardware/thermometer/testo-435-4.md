# Testo 435 4
| | | |:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------:|:-------------------------------------------------------------------------------------------------------------------------------------------------:| | [*Image: \1* | | | Status | supported | | Source code | [testo](http://github.com/OpenTraceLab/?p=OpenTraceCapture.git;a=tree;f=src/hardware/testo) | | Website | [testolimited.com](http://www.testolimited.com/testo-435-4-multi-function-instrument-for-vac-iaq) | **Testo 435-4** The **Testo 435-4** is a multifunction meter, capable of measuring airflow, humidity, temperature, indoor air quality, and more depending on the connected probes. It can store up to 10000 measurements internally, and communicates with a host via USB. See *Testo 435-4/Info* for more details (such as **lsusb -v** output) about the device. ## Hardware \- [Renesas M16C/62P (M30626FJPGP)](http://www.renesas.com/products/mpumcu/m16c/m16c60/m16c62p/device/M30626FJPGP.jsp) microcontroller. \- [Cypress CY62128DV30](http://datasheet.octopart.com/CY62128DV30LL-70SXI-Cypress-Semiconductor-datasheet-122470.pdf) 128K RAM chip. \- [FTDI FT232RQ](http://www.ftdichip.com/Products/ICs/FT232R.htm) serial-USB interface chip. ## Photos \-
