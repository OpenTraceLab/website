It is a clone of the [CWAV USBee DX](https://OpenTraceLab.org/w/index.php?title=CWAV_USBee_DX&action=edit&redlink=1 "CWAV USBee DX \(page does not exist\)"). See [HT USBee-DxPro/Info](HT_USBee-DxPro/Info.html "HT USBee-DxPro/Info") for some more details (such as **lsusb -v** output) on the device. **Note**: Due to the fact that this device has two FX2 chips behind a USB hub inside, this will need extra code to enumerate correctly in OpenTraceLab.
