## Attribution Requirements
### When Using OpenTraceLab
Please include appropriate attribution:
1. **Software attribution** - Include GPL license and copyright notices
2. **Documentation attribution** - Credit CC-BY-SA sources when adapting
3. **sigrok attribution** - Acknowledge sigrok project heritage
4. **Third-party attribution** - Include required third-party notices
### Attribution Format
Suggested attribution text:
```
Based on OpenTraceLab (https://opentracelab.github.io/website/),
which is derived from the sigrok project (https://sigrok.org/).
Licensed under GPL-3.0-or-later.
```
